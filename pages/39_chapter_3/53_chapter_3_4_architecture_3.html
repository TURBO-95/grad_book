The AI Kernel operates as the reasoning brain of the application. It utilizes a `ModelFactory` pattern to dynamically instantiate LangChain chat models, decoupling the system from any single provider constraint. Within the kernel, the `AgentRunner` maintains the asynchronous execution stream, while the `ActionEngine` resolves tool schemas, dynamically parses compiled `.proto` definitions for gRPC networks, and implements the human-in-the-loop pause mechanisms.
